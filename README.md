## ✨ 项目核心理念
「抽屉」不是为了让你写出完美的文章，而是为了在你灵感还热着的时候：
1. **全倒出来**：用极其简洁的对话界面，把脑子里的碎片倒给 AI。
2. **挖深抓牢**：AI 扮演速记员，通过精准追问帮你挖掘思维盲点。
3. **整理成卡**：聊偏了？AI 自动帮你提炼“核心想法”和“最小一步”。
4. **钉入待办**：让所有空谈最后都落实到一个具体的 Action。

## 🛠️ 技术特性
- **数据隐私**：所有点子本地存储 (`localStorage`)，服务器不留任何备份。
- **安全代理**：通过 Netlify Edge Functions 转发 AI 请求，API Key 永不暴露。
- **离线使用**：支持 PWA，可在 iPhone/Android 像 App 一样流畅使用。
- **动态图谱**：基于 D3.js 的实时思维演化图，不仅是笔记，更是生长的森林。

## 🚀 快速开始
1. 在 GitHub 上 Fork 本项目。
2. 部署至 Netlify。
3. 在 Netlify 后台设置 `SILICONFLOW_API_KEY` 环境变量。
4. 在手机浏览器中打开网址 -> “添加到主屏幕”。
